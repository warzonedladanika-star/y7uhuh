# BudgetBuddy aplikācija

## Apraksts
BudgetBuddy ir lietotājam draudzīgs budžeta plānotājs, kas palīdz sekot līdzi ienākumiem un izdevumiem. Lietotājs var pievienot ienākumus, izdevumus, skatīt statistiku un plānot nākotnes tēriņus.

---
